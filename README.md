### Topics Covered
- AWS Global Infrastructure

- Compute Services (EC2)

- Database Services (RDS, DynamoDB)

- `01-fundamentals/01-iam/README.md` - IAM deep dive

## AWS Certifications

1. **AWS Certified Cloud Practitioner** (CLF-C02)
   - Foundation level
   - ~3 weeks study
   - Location: `certifications/01-cloud-practitioner/`

2. **AWS Certified Solutions Architect - Associate** (SAA-C03)
   - Intermediate level
   - ~6 weeks study
   - Location: `certifications/02-solutions-architect-associate/`

3. **AWS Certified Developer - Associate** (DVA-C02)
   - Development focused
   - ~6 weeks study
   - Location: `certifications/03-developer-associate/`

4. **AWS Certified Solutions Architect - Professional** (SAP-C02)
   - Advanced level
   - ~8 weeks study
   - Location: `certifications/04-solutions-architect-professional/`

5. **AWS Certified DevOps Engineer - Professional** (DOP-C02)
   - Expert level
   - ~8 weeks study
   - Location: `certifications/05-devops-engineer-professional/`

## Getting Started

### Step 1: Set up AWS Account
- Create free AWS account at aws.amazon.com
- Set up IAM user with appropriate permissions
- Configure AWS CLI locally

### Step 2: Clone & Explore
```bash
git clone https://github.com/sumukham05-collab/main.git
cd main
```

### Step 3: Start Learning
- Begin with `01-fundamentals/01-iam/README.md`
- Follow the examples and complete the labs
- Move to the next topic once comfortable
