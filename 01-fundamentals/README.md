# Phase 1: AWS Fundamentals

Welcome to Phase 1 of your AWS learning journey! This phase covers the essential concepts and services you need to understand before moving to more advanced topics.

## What You'll Learn

- AWS Global Infrastructure
- Identity & Access Management (IAM)
- Elastic Compute Cloud (EC2)
- Simple Storage Service (S3)
- Relational & NoSQL Databases

## Learning Timeline: 3 Weeks

### Week 1: Core Concepts & IAM
- AWS Infrastructure overview
- IAM users, roles, and policies
- Security best practices

### Week 2: Compute & Storage
- EC2 instances and management
- S3 buckets and objects
- Pricing models

### Week 3: Databases & Integration
- RDS setup and management
- DynamoDB fundamentals
- Connecting services together
